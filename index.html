Concept Guide

개념은 비유로 잡으면 됩니다

Skill은 일하는 방식, MCP는 외부 도구로 들어가는 연결 통로, Command는 반복 요청을 줄이는 단축키입니다.

01

Skills

필요할 때만 꺼내 쓰는 전문 과외 선생입니다. 엑셀, PDF, 회의록, 제안서처럼 특정 업무가 들어오면 해당 절차와 기준을 불러옵니다.

02

MCP Servers

Notion, Figma, GitHub, Google Drive 같은 외부 도구로 들어가는 열쇠입니다. 연결 없이는 AI가 바깥 자료를 직접 볼 수 없습니다.

03

Slash Commands

자주 쓰는 긴 요청을 한 줄로 줄인 단축키입니다. 매번 같은 설명을 반복하지 않도록 도와줍니다.

Build Your Skill

나만의 Skill 제작 흐름

처음부터 Skill을 만들지 말고, AI와 일을 한 번 끝까지 해본 뒤 그 과정을 절차화하는 방식이 가장 자연스럽습니다.

1반복성

반복 업무 고르기

3번 이상 반복했고 입력자료와 결과물 형태가 분명한 일을 고릅니다.

2실행

AI와 끝까지 해보기

회의록 정리, 리서치 요약, 제안서 초안처럼 실제 업무 하나를 처리합니다.

3형식화

기준을 Skill로 바꾸기

작업 순서, 판단 기준, 출력 형식, 실패 사례를 정리합니다.

AI에게 이렇게 말해보기 
방금 우리가 한 작업을 그대로 Skill로 만들어줘.
다음에 비슷한 요청이 오면 같은 절차로 처리할 수 있도록,
작업 순서, 판단 기준, 출력 형식, 주의사항을 정리해줘.

공지에 붙일 문구
